#deltavistoria_projeto2

Integrantes: Hiago Silva, Ryan Lustosa, João Vitor Fujarra, Fernando Wenner, Gutemberg Cezar.
Projeto: Site Delta Vistoria

Objetivos:
Implementação da estrutura de agendar vistorias, cadastrar clientes, orçamentos e um painel administrativo utilizando php onde será possível ter todo controle de edição dos clientes e vizualização das ordens de serviços.

A aplicação web será desenvlvida para um uma empresa real situada nas regiões metropolitanas de João Pessoa, onde a mesma presta serviços de vistoria veicular, vistoria cautelar e transferencia de proprietário. O objetivo da aplicação é ingressar a empresa na web tendo em vista que atualmente utiliza apenas das redes sociais como meio de se comunicar com os clientes. Através da aplicação o cliente poderá conhecer mais a empresa, agendar vistorias ou qualquer outro serviço prestado e realizar um prévio orçamento, também será implementado um painel administrativo de acesso restrito, onde o admin poderá ter acesso aos agendamentos feito pelos clientes, criar novos agendamentos, criar novos clientes e editar ou excluir ambos.
